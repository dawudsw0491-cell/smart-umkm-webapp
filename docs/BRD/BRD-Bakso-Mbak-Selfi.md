# 1. Latar Belakang

Bakso Mbak Selfi merupakan usaha kuliner yang berlokasi di depan pintu masuk Candi Tegowangi, Desa Tegowangi, Kecamatan Plemahan, Kabupaten Kediri, Jawa Timur.

Usaha ini memerlukan sistem berbasis web yang dapat membantu operasional usaha sekaligus menjadi media promosi digital.

---

# 2. Tujuan Project

Membangun aplikasi berbasis web yang mampu:

* Menampilkan informasi usaha.
* Menampilkan menu makanan dan minuman.
* Menampilkan promo.
* Menampilkan lokasi usaha.
* Mendukung operasional kasir.
* Mendukung pembukuan sederhana.
* Mendukung CRM sederhana.

---

# 3. Ruang Lingkup

* Website Publik
* Portal Owner
* Portal Kasir
