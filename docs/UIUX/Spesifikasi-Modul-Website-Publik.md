# 1. Tujuan

Modul Website Publik berfungsi sebagai media informasi dan promosi digital yang dapat diakses oleh pelanggan tanpa proses login.

---

# 2. Target Pengguna

Pengguna utama:

* Pelanggan
* Wisatawan
* Masyarakat umum

---

# 3. Struktur Menu

Website terdiri dari menu berikut:

1. Beranda
2. Menu
3. Promo
4. Wisata Sekitar
5. Galeri
6. Lokasi
7. Hubungi Kami

---

# 4. Halaman Beranda

Menampilkan:

* Logo usaha
* Nama usaha
* Banner utama
* Deskripsi singkat usaha
* Tombol Lihat Menu
* Tombol Promo
* Tombol WhatsApp
* Tombol Google Maps
* Menu favorit
* Promo aktif
* Destinasi wisata sekitar
* Footer

---

# 5. Halaman Menu

Menampilkan:

* Foto menu
* Nama menu
* Kategori
* Harga
* Status ketersediaan

---

# 6. Halaman Promo

Menampilkan:

* Banner promo
* Nama promo
* Deskripsi promo
* Periode promo

---

# 7. Halaman Wisata Sekitar

Menampilkan:

* Foto destinasi
* Nama destinasi
* Deskripsi singkat
* Tombol Google Maps

---

# 8. Halaman Galeri

Menampilkan:

* Foto makanan
* Foto minuman
* Foto tempat usaha

---

# 9. Halaman Lokasi

Menampilkan:

* Google Maps
* Alamat lengkap
* Jam operasional
* Tombol Navigasi

---

# 10. Halaman Hubungi Kami

Menampilkan:

* Nomor WhatsApp
* Tombol Chat WhatsApp
* Instagram
* TikTok

---

# 11. Navigasi

Urutan menu:

* Beranda
* Menu
* Promo
* Wisata Sekitar
* Galeri
* Lokasi
* Hubungi Kami

---

# 12. Ketentuan

* Tidak memerlukan login.
* Seluruh halaman dapat diakses oleh publik.
* Tampilan responsif pada perangkat desktop dan mobile.
* Seluruh teks menggunakan Bahasa Indonesia.

---

# 13. Kriteria Selesai

Modul Website Publik dinyatakan selesai apabila seluruh halaman dapat diakses sesuai struktur menu dan menampilkan informasi yang telah ditentukan.
